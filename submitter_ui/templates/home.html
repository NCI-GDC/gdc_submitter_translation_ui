GDC Submitter Translation UI

This application is to help submitter create YAML and CONF input files for the GDC submitter.

Check more details on About page before start!

START

    07/03/2018 Version 2.0:

  • The UI can accept multiple nodes from single uploaded TSV, which will generate multiple node yaml files and one single task conf file and one to_be_deleted.yaml if desired.

    05/29/2018 Version 1.0:

  • The UI is able to create one single node from uploaded TSV at a time.

  • The CONF file could only have ONE link. However the UI is able to create multiple links if submitter selects more than one.
